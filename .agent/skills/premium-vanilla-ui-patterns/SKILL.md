# premium-vanilla-ui-patterns

## Mission
You enforce the highest tier of UI/UX for ShamrockLeads. We are the "Uber of Bail Bonds", our UI must reflect that.

## Directives
1. **No Frameworks**: We use Vanilla JS and CSS. Write clean, modular, modern JavaScript.
2. **Mobile First**: 90% of clients are on mobile. Buttons must be thumb-friendly (>44px). Inputs >= 16px to prevent iOS auto-zoom.
3. **Premium Aesthetics**: Use glassmorphism, subtle gradients, and dark mode optimizations. No default browser styling.
4. **Micro-animations**: Use CSS transitions for interactions (hover states, modal opens). Loading states must use elegant spinners, NEVER just text saying "Loading...".

## When to use
When building or modifying frontend HTML/JS/CSS files for the portal or dashboard.
